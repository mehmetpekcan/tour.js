This guide sets you up to create your own tours

#### Install

Install through cdn:

```bash
https://cdn.jsdelivr.net/npm/tour.js@0.1/dist/index.min.js

https://cdn.jsdelivr.net/npm/tour.js@0.1/dist/style.min.css
```

Using `npm`:

```bash
npm install —save tour.js
```

Using `yarn`:

```bash
yarn add tour.js
```

#### Usage

Tour.js consists of js and CSS files. The CSS file must be linked/imported throughout your app.
If you use any bundler for your app you can use it directly by importing in your js file.

```javascript
import Tour from ‘tour.js’
```

First crate a new `Tour` instance for your tours with steps.

```javascript
const tour = new Tour({
	/// Tour options will be here
	steps: [
		{
			selector: #my-greate-title
			title: ‘Learn how to use tour.js’
			content: ‘Here this is the great explanation for showing how to use tour.js in real world project!’
		},
		// add more steps here
	]
});
```

Then fire it whenever you wish

```javascript
tour.start();
```

Alternatively you could manipulate the steps any time even that you can create instance without explicitly giving the steps while initializing.

```javascript
tour.changeSteps([
  /// steps array will be here
]);
```

#### Step Api

<section className="inner-section">

##### selector

Selector of the element that you want to highlight. To see how to get any element of selector in any website please read [here](https://support.helpdocs.io/article/eh3pzxNwYf-grabbing-css-selectors-from-your-browser#:~:text=Simply%20right%20click%20and%20click,CSS%20selectors%20for%20that%20element.)

</section>
<section className="inner-section">

##### target

`target` is an alternative of `selector` property. You could give the `target` element directly without specifying its selector.

```javascript
const targetEl = document.getElementById(‘#my-great-element’);
const tour = new Tour({
	steps: [
		{
			target: targetEl,
			// other properties will be here
		}
	]
});
```

</section>

<section className="inner-section">
  !Note that if you provide `selector` and `target` both in a step. `selector`
  will be used for getting the element by Tour.js.
</section>

##### content

Body of your step element. It could be just text or any HTML content that you provide.

```javascript
This is a valid content text.

<p>
	This is also valid <strong>context</strong>.
</p>

<div>
	<h2>
		This is even valid <i>context</I>
	<h2/>
	<p>
		This is an extra element.
	</p>
</div>
```

```javascript
const tour = new Tour({
  steps: [
    {
      next: false, // `next` button will not be disappear in this individual step.
      // rest of your properties
    },
  ],
});
```
